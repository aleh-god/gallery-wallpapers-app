## Gallery wallpapers Application

Interview task.
It is required to develop an Android application in Kotlin, through which you can set wallpapers on your phone's desktop.

**Technologies used:**

1. RecyclerView
2. Glide
3. Retrofit
4. Hilt
5. Coroutines - Flow
6. Clean Architecture, MVVM pattern.
